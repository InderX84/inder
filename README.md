# Narinder Singh — Portfolio

Personal portfolio website built with React and Vite, showcasing projects, skills, education, certifications, and contact information.

## Features

- Glassmorphism design with animated gradient background and floating orbs
- Inverted pyramid skills section with brand-colored technology icons
- GitHub projects section with live search, language filter, sort, pagination, and live/repo links
- GitHub activity section with language usage breakdown
- Accordion-style certificates section with in-app PDF preview modal
- About section with education timeline
- Achievements & Currently Learning section with tab switcher
- Contact form powered by EmailJS
- Resume preview modal with download option
- Scroll progress bar and back-to-top button
- 404 page with client-side routing
- SEO meta tags, Open Graph, and Twitter Card support
- Fully responsive across mobile, tablet, and desktop

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- React Icons
- Axios
- EmailJS
- React Router DOM
